# Data use notice

This public release is ID-only. It contains X post identifiers together with
researcher-created labels, domain assignments, structured annotations, split
memberships, predictions, and aggregate results. It does not redistribute
hydrated post text, post URLs, account identifiers, profile descriptions,
timestamps, engagement/profile snapshots, row-level embeddings, or free-text
LLM rationales.

Possession of an ID does not confer rights to obtain, publish, retain, or use
the associated content. Users are responsible for their own X API access and
for complying with applicable platform terms, law, ethics requirements, and
institutional policies. Content that is deleted, protected, suspended, or
otherwise unavailable should not be reconstructed from third-party copies.

Locally hydrated data are written to `data/generated/`, which is excluded from
version control. Do not commit or redistribute that directory as part of this
release.

Requests concerning researcher-created annotations or labels should identify
the relevant `record_id` or `tweet_id` and be directed to the repository
maintainers. Platform-content requests should be handled consistently with the
current state of the underlying X post and applicable obligations.
